**apt Paket Sistemi**
+++++++++++++++++++++

apt paket sistemi debian tabanlı sistemlerde paket yönetimi için kullanılan bir uygulamadır.
Temel işlemler şunlardır.

Yerel Paket İndexini Güncelleme
-------------------------------

.. code-block:: shell

	sudo apt  update

Paket Yükleme
-------------

.. code-block:: shell

	sudo apt install paket_adi

Paket Silme
-----------

.. code-block:: shell

	sudo apt remove paket_adi
